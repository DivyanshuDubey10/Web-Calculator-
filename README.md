# Web Calculator 🧮

A sleek, web-based calculator built with HTML, CSS, and JavaScript. This was created as a semester project for my BCA program.

## Features
* Performs basic arithmetic operations (+, -, *, /).
* Features a modern, mobile-app style user interface.
* Error handling for division by zero and invalid inputs.
